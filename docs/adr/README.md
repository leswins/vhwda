# Decision Log (ADRs)

We track architectural decisions in lightweight ADRs (Architecture Decision Records).

## Format
- `NNNN-short-title.md` (e.g. `0001-monorepo-structure.md`)
- Keep each ADR short:
  - Context
  - Decision
  - Consequences
